Ce sur quoi on ne négocie pas

Gratuit, vraiment

Pas de version premium. Pas de limite cachée. Ce que vous voyez, c'est ce que vous avez.

Confidentiel par design

Vos données ne transitent pas par nos serveurs. On ne peut pas les voir, même si on le voulait.

Sécurité maximale

Double couche de protection. Clés éphémères. Les mêmes standards que les pros utilisent.

Transparent

Code ouvert. Chaque ligne est vérifiable. On n'a rien à cacher parce qu'on n'a accès à rien.
